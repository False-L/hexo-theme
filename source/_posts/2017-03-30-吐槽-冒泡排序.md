## 补上一个冒泡排序
```javascript
var A=[12,5,7,1,4,13,10,10,13,1,0,5,9];
var temp;
for(var i=0;i<A.length;i++){
    for(var j=A.length-1;j>i;j--){
        if(A[j]<A[j-1]){
            temp=A[j];
            A[j]=A[j-1];
            A[j-1]=temp;
        }
    }
}
console.log(A);
```
